# React Enquiry Form Application

This project is a fully functional React based enquiry form application designed to collect user information such as name, email, phone number and message. The submitted details are displayed instantly in a dynamic table where users can edit or delete any record. The application uses React controlled components and state management to handle form inputs efficiently and provides toast notifications for better user interaction.

The layout is styled using Bootstrap to maintain a clean, responsive and user-friendly interface across different screen sizes.

## Features

• Add new enquiry records  
• View submitted data in real time  
• Edit existing records  
• Delete unwanted entries  
• Toast notifications for user actions  
• Responsive form layout using Bootstrap  

## Technologies Used

React JS, JavaScript, Bootstrap, React Toastify, HTML, CSS

## Installation and Setup

Clone the repository and navigate to the project folder, then run the following commands:

npm install  
npm start  

The application will start at http://localhost:3000

## Project Purpose

This project demonstrates the practical use of React hooks, controlled form components, dynamic rendering of data, and basic CRUD operations in a real world frontend application.
